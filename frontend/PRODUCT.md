# TestCaseIQ — Product Context

**Product:** AI-powered QA test case management platform  
**Audience:** QA engineers and software teams managing test coverage for complex applications  
**Core jobs:** Write and organize user stories, generate AI test cases, review and approve test coverage, export to Playwright / Postman / Jira Xray / Azure DevOps  
**Design register:** App UI (design serves the product, not the product itself)  
**Theme:** Black Glass Instrument — deep dark (#0a0a0a), phosphor accent (#b8ff5a), cyan/purple/green/amber semantic signals  
**Typography:** Syne display (h1–h2) + Inter body + JetBrains Mono code  
**Motion:** Lenis smooth scroll, GSAP entrance animations, CSS view transitions  
**Stack:** Angular 22, Spring Boot, PostgreSQL  
**Tone:** Precise, technical, premium — a tool serious engineers trust
